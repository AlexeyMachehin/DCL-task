## DCL-task

Ссылка на netlify.com: [DCL-task](https://dcl-task.netlify.app)

## Стек технологий: 
* Angular 
* Eslint
* Stylelint

## Запуск dev сборки:
Ввести в терминале команды:
* git clone https://github.com/AlexeyMachehin/DCL-task.git
* npm i
* npm run start

Production сборка - npm run build
